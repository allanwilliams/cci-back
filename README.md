- [Postgres](https://www.postgresql.org/) - 14.0.*

## Instalação
* instale o venv
    python3 -m pip install --user virtualenv
* vá para a pasta do projeto e crie o venv
    python -m venv venv
    
* habilite a venv
    source venv/bin/activate

* instale os requirements
pip install -r requirements.txt
